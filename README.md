## Installation

1. Open the project in VS Code.
2. Run Schema.sql against your server
3. Configure the server connection in <a href="db.js">db.js</a> lines 4 through 8
4. In VS Code Explorer Open NPM Scripts Tab
5. Run seed
6. Run start
